# References — Index

Список внешних проектов/материалов, которые используем как референсы (идеи/паттерны/UX).

Важно:
- Перед копированием кода **всегда** проверяем лицензию.
- Если проект под GPL/AGPL — берём только идеи/паттерны, **не копируем код** в MIT‑проект.
- Если нужно быстро выбрать референсы “под фичу” — начать с `docs/references/ROADMAP.md`.

Как добавлять новый референс:
1) Создать заметку по шаблону: `docs/references/projects/_template.md`
2) Добавить строку в таблицу ниже
