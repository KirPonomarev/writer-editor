# Phosphor toolbar icons

This directory contains a bounded subset of Phosphor Icons Regular SVG assets used by the Yalken formatting toolbar.

- Style: Regular
- Upstream copyright: 2020-2024 Phosphor Icons
- License: MIT, included in `LICENSE`
- Integration: local CSS masks using `currentColor`

Only icons used by the live toolbar are bundled. The upstream archive is not copied into the application.
